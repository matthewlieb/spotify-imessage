First time opening? Anyone who downloads the app from the web may need to do this once—macOS adds a security flag we can't remove without Apple Developer signing. Try Right‑click → Open first; if macOS says "damaged", use Terminal:
xattr -dr com.apple.quarantine ~/Downloads/SpotifiMessage.app
If you moved it to Applications:
xattr -dr com.apple.quarantine /Applications/SpotifiMessage.app
Open Terminal (Spotlight → "Terminal"), paste, press Enter, then double‑click the app. The full build includes Python; see the release notes for more.
